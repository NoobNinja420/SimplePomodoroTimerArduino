There is no audio, but you can use it for simple timer example.
All you need is LCD 16x2, 256 om resistor,Potentiometer 10k om to adjust brightness of text, 4x4 buttons some cables and of course arduino main board.
i connect LCD without i2c module.
this timer has 4 presets 50mins, 25 mins, 10 mins, 5mins. you can edit presets in code.
